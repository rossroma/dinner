# 逍邦网络点餐系统	

## Note

1. 请谨慎填写点餐人姓名，一经填写即视为默认点餐人，并存储到LocalStorage中；
2. 移动端适配，目前只兼容iPhone Plus 机型；
3. 普通点餐者只能撤销自己的点餐；
4. 右上角为管理员登录入口，登陆成功后，可开启/关闭点餐状态，筛选历史记录，撤销任何人的点餐、设置点餐白名单、编辑食谱等；
